The project is supposed to implement the communication among different devices.
First of all, there should be a device runs the server program. To run the server, please start command from THIS directry and type:

[user@path]# cd server/out/production/server/
[user@path]# java server.server_main_structure

And then, you can start many client programs from different devices (make sure they are in the same network and there is no blockade of connection between devices from the network.
